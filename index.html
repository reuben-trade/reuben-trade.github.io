The centrepiece

Machine Learning Engineer · WaveX

Machine learning for wave energy.

At WaveX, an Australian wave-energy company, I'm developing an intelligent, learning control system to maximise power generation in the highly nonlinear environment of ocean waves, across a converter of multiple independent bodies that move and couple together. I'm also building the 3D simulation environment used to visualise the system and the data flowing through it. WaveX is sponsoring my honours project.

Featured · Research project

A brain-inspired operating system for drones.

A control stack for drones operating in remote, GPS-denied environments. Each module is inspired by a different region of the brain, mirroring how a biological system senses, remembers, decides and moves, so the drone can perceive its surroundings and act on its own the moment conditions change, instead of waiting on a link back to base. I'm exploring reinforcement learning to sharpen how it adapts.
